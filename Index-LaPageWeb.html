Une page web est créé par trois langage informatique different qui on chaqu'une utilité:
  • Le premier est l'Html qui sers a créé le texte et les liens qui s'y trouve.

  • Le deuxieme est le CSS qui a pour but d'embelir les pages avec des images ect

  • Le troisieme est le JavaScript qui sers a faire les interaction par l'utilisateur sur la page.
