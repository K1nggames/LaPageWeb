Bienvenue sur la partie html de LaPageWeb

L'html est le langage utiliser pour coder le texte et les liens a l'interieur.
Pour commencer tout dossier en html il faut mettre un <!DOCTYPE html> pour precisez que c'est un document en html.
Si vous voulez definir en quel langue a était fait la page, vous pouvez mettre
<html lang=""> ,puis mettez la lange entre les gimets.

Pour après il faudrat mettre le corp de text, il daudrat donc mettre des <head> <body>.

On va donc pouvoir commencer à ecrire le texte pour la page. Tout cela se fait avec des attribue.
Les attribues son des lettres entouré de cheuvrons ( < > ).
Les attribues se mettent au debut et en fin de text avec un / rajouter.:

Ex: <p> Je suis un text <p>

Voici la liste des attibuts les plus important:

  • <p> </p> est simplement les attibuts pour mettre un text.

  • <strong> </ strong> est a mettre autour d'un texte pour le mettre en gras,
    ce seras pareil avec <em> </em> pour mettre en Italic et <u> </u> pour souligner.

  • <h?> </h?> il faudrat remplacer le "?" par un chiffre pour valoriser un text en gros.
    Plus le chiffre se raproche de 0, plus le text seras gros.


    • Lien video:https://youtu.be/LkZ-KqxgVQ4

Après avoir écrie le texte, on pourras rajouter des liens avec l'attribue <a href="">text</a>.
Il faudrat rajouter le lien entre les deux "" et mettre le texte qui servira de lien au lieu du "text".

Ex: <a href="Index-LaPageWeb.html">Accueil</a>

Les lien de ce type vont alors remplacer la page actuel par une autre,
mais si vous voulez que la page s'ouvre dans un nouvel onglet il faudrat rajouter:
target"blank_" après les deux gimets et avant le >.

Ex:<sa href="https://grafikart.fr/formations/html" target="_blank">Grafikart html>/a>
