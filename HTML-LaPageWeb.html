Bienvenue sur la partie html de LaPageWeb

L'html est le langage utiliser pour coder le texte et les liens a l'interieur.
Pour commencer tout dossier en html il faut mettre un <!DOCTYPE html> pour precisez que c'est un document en html.
Si vous voulez definir en quel langue a était fait la page, vous pouvez mettre
<html lang=""> ,puis mettez la lange entre les gimets.

Pour après il faudrat mettre le corp de text, il daudrat donc mettre des <head> <body>.

On va donc pouvoir commencer à ecrire le texte pour la page. Tout cela se fait avec des attribue.
Les attribues son des lettres entouré de cheuvrons ( < > ).
Les attribues se mettent au debut et en fin de text avec un / rajouter.:

Ex: <p> Je suis un text <p>

Les attributs (Balise):

  • <p> </p> est simplement les attibuts pour mettre un text.

  • <strong> </ strong> est a mettre autour d'un texte pour le mettre en gras,
    ce seras pareil avec <em> </em> pour mettre en Italic et <u> </u> pour souligner.

  • <h?> </h?> il faudrat remplacer le "?" par un chiffre pour valoriser un text en gros.
    Plus le chiffre se raproche de 0, plus le text seras gros.


    • Lien video:https://youtu.be/LkZ-KqxgVQ4

Les liens:

Après avoir écrie le texte, on pourras rajouter des liens avec l'attribue <a href="">text</a>.
Il faudrat rajouter le lien entre les deux "" et mettre le texte qui servira de lien au lieu du "text".

Ex: <a href="Index-LaPageWeb.html">Accueil</a>

Les lien de ce type vont alors remplacer la page actuel par une autre,
mais si vous voulez que la page s'ouvre dans un nouvel onglet il faudrat rajouter:
target"blank_" après les deux gimets et avant le >.

Ex:<sa href="https://grafikart.fr/formations/html" target="_blank">Grafikart html>/a>

Pour faire un lien qui redirige directement vien un endroit de votre site, vous devez créer une id.
<a href="#ID"> Texte 1</a> , vous mettez un nom a la place de ID que vous réutuliserez après avec votre texte après,
puis, plus tard dans votre page web, vous pourrez mette,
<p id="ID" >Texte 2 </id>, vous remmetez l'ID que vous avez mis pour le premier. Au final, quand vous clicerez sur le lien
cela vous emmenneras automatiquement à l'endroit demandez.

Ex:<a href="#Probleme"> Question ?</a>

<p id="Probleme" >Reponse </id>

Les images:

Avec l'html, on peut aussi y placer des images dont on peut changer de dimension.
On peut mettre des images de plusieurs format (Png,Jpg,Svg,Gif...).
Vous pouver donc mettre une image en mettant <img src="chemin"alt="nom">.
Il faudrat copier votre image dans votre editeur de code et mettre à la place de "chemin",
le chemin des fichier pour arriver jusqu'à votre image en n'oubliant pas de mettre en quel format est l'image.
Puis vous metterez un nom après alt= entre les gimets.
Pour redmiensionnez l'image, il faudrat mettre height="" et withd="" après le nom que vous avez donnez à l'image.
Il ne vous resteras plus qu'à mettre les dimensions entre les gimet ( "" ).
Si vous voulez redimmensionnez à égalitez, mettez seulement l'une des deux dimensions choisi.

Ex:<img src="img/html.png"alt="html" width="100"> Si vous voulez faire une image qui redirige vers un lien quand on lui clic dessus,
Vous devrez ré-utiliser le code de la parite Les liens mais vous metterez le /a> après l'image à afficher. <a href="lien" target_blank"> >img src="htlm.png" alt="htlm"> >/a> Lien video:https://youtu.be/7T9H-P45REg
